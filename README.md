# VelOrda - Bicycle Rental Admin Panel

## Tech Stack
- Backend: Node.js + Express + MongoDB
- Frontend: React + Vite + Tailwind CSS

## Setup & Run

### 1. Install dependencies
cd server && npm install
cd ../client && npm install

### 2. Seed the database
cd server
node seed.js

### 3. Start backend
cd server
npm run dev

### 4. Start frontend
cd client
npm run dev

### 5. Open browser
http://localhost:5173
